
# 📊 Dashboard de Inflação

# Este projeto possui um dashboard interativo no Google Sheets, atualizado automaticamente via pipeline ETL.

🔗 Acesse o dashboard:
https://docs.google.com/spreadsheets/d/1J0tqH7Lc_5n6m3lnIspF6KK1LNZLJJP6yY4VXWRfLZY/edit?usp=sharing

## 📌 O que você encontrará:
## - Evolução da inflação ao longo do tempo
## - Média móvel de 3 meses
## - Ranking de anos com maior inflação
## - Análise mensal

## 🔄 Atualização
Os dados são atualizados automaticamente através de um pipeline em Python com execução agendada.
